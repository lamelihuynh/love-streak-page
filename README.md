# Love Streak Page

Trang web tĩnh để host lên GitHub Pages.

## Cách dùng

1. Đưa toàn bộ thư mục `love-streak-page` lên GitHub.
2. Vào repository trên GitHub.
3. Chọn `Settings` > `Pages`.
4. Ở phần `Build and deployment`, chọn deploy từ branch chứa file `index.html`.
5. Mở link GitHub Pages và gửi cho bạn gái.

Nếu muốn sửa lời nhắn, chỉnh nội dung trong `index.html`.
